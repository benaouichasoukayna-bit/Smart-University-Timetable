#  Smart University Timetable Manager

## Présentation
Une application intelligente développée en **Python (POO)** permettant la génération automatique et l'optimisation des emplois du temps universitaires.

##  Fonctionnalités Clés
- **Génération Automatique :** Création de plannings sous contraintes.
- **Multi-Profils :** Interfaces pour Administrateurs, Enseignants et Étudiants.
- **Détection de conflits :** Algorithme intelligent pour éviter les chevauchements.
- **Export :** Génération de rapports PDF et Excel.

##  Technologies
- Langage : Python 3
- Paradigme : Programmation Orientée Objet (POO)
